 Amigo Secreto
 _______________________________________________________________________________________

Um projeto em HTML, CSS e JavaScript para deixar o sorteio de amigos mais divertido!
O usuário pode montar uma lista de participantes e  sortear um nome de forma aleatória.

________________________________________________________________________________________
O que dá pra fazer

Cadastrar nomes de amigos em uma lista.
Visualizar em tempo real os participantes adicionados.
Realizar um sorteio rápido e aleatório.

________________________________________________________________________________________
Tecnologias 

HTML5
CSS3
JavaScript
